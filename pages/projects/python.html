Amazon Product Price Tracker

Project Overview: A simple Python-based tool that scrapes product information from Amazon (title, price, and date), stores it in a CSV file, and allows you to track price changes over time.

Features:

  • Scrapes product title and price from an Amazon product page
  • Cleans and structures the data for readability
  • Saves the product information into a CSV file
  • Appends new data entries over time to build a price history
  • Allows automated price checks at custom time intervals

Technologies Used:

  • Python 3 with BeautifulSoup for HTML parsing
  • Requests for fetching web pages
  • Pandas for data handling and analysis
  • CSV/OS/Time/Datetime for file management & scheduling
